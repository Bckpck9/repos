Прямоугольник – периметр, площадь, длина диагонали.
Результат работы программы
Меню выбора:
Программа предлагает пользователю выбрать, что нужно вычислить: периметр, площадь, длину диагонали или все сразу.
Выбор реализован через switch.
Ввод данных:
В каждом блоке case программа запрашивает у пользователя длины сторон a и b.
Вызов функции:
В зависимости от выбора пользователя программа вызывает соответствующую функцию:
perimetr — для вычисления периметра.
Периметр: рассчитывается по формуле 2 * (a + b).
square — для вычисления площади.
Площадь: рассчитывается по формуле a * b.
lengthDiagonal — для вычисления длины диагонали.
Длина диагонали: рассчитывается по формуле sqrt(a^2 + b^2).
Если выбран вариант 4, программа вызывает все три функции.
Все вместе: Программа выводит все три значения: периметр, площадь и длину диагонали.

Треугольник – периметр, площадь по формуле Герона, проверка на равнобедренность.
Результат работы программы
Ввод данных:
Программа запрашивает у пользователя длины сторон треугольника a, b и c.
Проверка существования треугольника:
Программа проверяет, существует ли треугольник с такими сторонами, используя условие: a + b > c && a + c > b && b + c > a
Если треугольник не существует, программа выводит сообщение: “Треугольник с такими сторонами не существует.”
Меню выбора:
Если треугольник существует, программа предлагает пользователю выбрать, что нужно вычислить: периметр, площадь, проверить на равнобедренность или все сразу.
Выбор реализован через switch.
Вызов функции:
В зависимости от выбора пользователя программа вызывает соответствующую функцию:
perimeter — для вычисления периметра.
Периметр = a + b + c
plosh — для вычисления площади по формуле Герона.
Площадь = sqrt(s * (s - a) * (s - b) * (s - c)), где полупериметр треугольника, вычисляемый как: s = (a + b + c) / 2
ravnob — для проверки на равнобедренность.
Треугольник является равнобедренным, если хотя бы две его стороны равны: a == b  b == c  a == c
Если выбран вариант 4, программа вызывает все три функции.
Результат вычислений выводится на экран.

Трапеция – периметр, площадь, длина средней линии.
Результат работы программы
Меню выбора:
Программа предлагает пользователю выбрать, что нужно вычислить: периметр, площадь или среднюю линию трапеции.
Выбор реализован через switch.
Ввод данных:
В зависимости от выбора пользователя программа запрашивает необходимые данные:
Для периметра: длины всех четырех сторон трапеции (a, b, c, d).
Для площади: длины двух оснований (a, b) и высоту (h).
Для средней линии: длины двух оснований (a, b).
Вызов функции:
В зависимости от выбора пользователя программа вызывает соответствующую функцию:
perimeter — для вычисления периметра.
Периметр трапеции равен сумме длин всех ее сторон: a + b + c + d
area — для вычисления площади.
Площадь трапеции вычисляется по формуле: ((a + b) / 2) * h , где 
a, b — длины оснований трапеции.
h — высота трапеции.
midle_line — для вычисления средней линии.
Средняя линия трапеции равна полусумме длин оснований: (a + b) / 2
Результат вычислений выводится на экран.
